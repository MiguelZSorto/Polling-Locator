About

Polling Locator was developed for CSCE 315 Software Engineering Studio at Texas A&M University

Technologies Used

The frontned is built with HTML, CSS, and JavaScript. We use Bootstrap as the frontend framework to give the applications its mobile friendly design.

The backend is built with Node.js and uses the Express and Request modules. We use the Google Civic API to retrieve data about the voting locations around a given address, Here WeGo Maps to plot those locations for the user, and we use Who Is My Representative to gather the list of representatives that represent the given address.
